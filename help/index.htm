The game won't show up

For Pong, make sure you have JavaScript enabled. If you're using Chrome, you can check by going to Chrome's JavaScript settings. Make sure this site is not on the "Block" list. Firefox will always allow Javascript.

All other games are either board games (just instructions and a board) or executable JAR files that have to be downloaded.

I can't start the game

Pong

Click the mouse to start the game. Click it again to pause.

Downloaded games

Click "Run" to start the game.

I don't know how to play

Go to the index page, then select the game you want to play.
