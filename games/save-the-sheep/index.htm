Save the Sheep!


Controls

Move car up, down, left, right
Add more chickens C
Shoot Space

Objective

The objective of this game is to shoot/drive over ("kill") 3 robots while there are still sheep. If you (the car) park on a gas pump, you will get more fuel. If you run out of fuel, you can shoot a gas can and drive to a pump.
