The game won't show up

For Pong, make sure you have JavaScript enabled. If you're using Chrome, you can check by going to the settings. Make sure this site is not on the "Block" list. Firefox will always allow Javascript.

All other games are either board games (just instructions and a board) or executable JAR files that have to be downloaded.

I can't start the game

Pong

Click the mouse to start the game. Click it again to pause.

Downloaded games

Click "Run" to start the game. Java doesn't come with Windows, so Windows users can download Java here. Depending on how you installed Java, you might be able to double-click on a JAR file, wait a few seconds, click Run, and start the game. If that doesn't work, you can follow these steps:
Click to show steps
  1. In the file browser, right-click your "Computer" or "This PC" folder and choose "Properties".
  2. Click "Advanced system settings" in the left pane.
  3. Click "Environment Variables"
  4. In the "System Variables" section, select "Path" and press the "Edit" button.
  5. At the beginning of the text box, add the path to the folder where Java is, followed by a semicolon. This will allow you to run Java in your command prompt.
  6. Click the OK button on the Edit System Variable, Environment Variables, and System Properties dialog boxes. You can test if it worked by typing "java -version" in your command prompt.
  7. Try double-clicking on the JAR file again. If it doesn't work, open your command prompt, type "java -jar ", followed by the location of the JAR file, like this:

    java -jar C:\Users\ME\Desktop\invaders.jar

    Replace "ME" with your username. If the path has spaces in it, surround it with quotes.
  8. Click "Run" to play the game

How to open ZIP files
Windows

Right-click on the file, then click "Extract All...". You will come to a screen that looks like this:

Select a destination for the files (by default it will place them in a new folder inside the folder that contains the ZIP file), then click "Extract".

Ubuntu

Double-click on the file. The Archive Manager will open. Click "Extract". Screenshot coming soon.
